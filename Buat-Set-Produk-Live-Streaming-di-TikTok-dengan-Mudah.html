Buat Set Produk Live Streaming di TikTok dengan Mudah

Ada beberapa hal yang perlu kamu persiapkan sebelum menjalankan live streaming, salah satunya adalah menyiapkan produk-produk yang akan ditayangkan selama live streaming. Dan saat ini kamu bisa dengan mudah mengatur urutan produk sebelum live dimulai.​
​
​
Fitur ini disebut dengan Set Produk Live, dimana kamu bisa dengan mudah mengatur daftar produk live. Fitur ini sangat bermanfaat untuk kamu ​
  • Yang memiliki jumlah produk yang banyak ​
  • Rutin melakukan live dengan jenis produk yang sama​
​
Menarik kan? Yuk, mulai setting daftar produk live kamu agar sesi live streaming semakin maksimal!​
​
FAQ:​
  1. Apakah ada batasan jumlah produk dalam Set Produk LIVE?​
Kamu bisa memasukan jumlah produk set dengan maksimal 50 produk.​
  1. Bisakah saya mengubah urutan set produk?​
Kamu bisa mengubah urutan produk (menyeret, memindahkan ke atas, memindahkan ke atas secara massal, mengedit peringkat produk) pada halaman Seller Center.​
